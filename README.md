# Give Feedback Publicly

**Give Feedback Publicly** is a lightweight public feedback board where anyone can browse sample feedback, filter by popular hashtags, and submit new feedback through a form.

This project is a **public demo/portfolio** for showcasing ideas and UI/UX behavior. It is built with **Vanilla JavaScript** and **Tailwind CSS**, powered by **Vite**.

## Features

- Public feedback board (sample feedback included)
- Add new feedback via a form (new feedback gets added to the dataset)
- Clickable hashtags
- View feedback details: click a feedback card to open a detailed view
- Hashtag browsing:
    - Click a hashtag inside a feedback to view related items
- Sort feedback by:
    - **Date**
    - **Views/Visit count**
- Optional control: **disable a selected hashtag**
- Popular tags section (based on hashtag usage)

## Tech Stack

- **JavaScript (Vanilla)**
- **Tailwind CSS**
- **Vite**

## Demo / Preview

Open the project and run the development server:

```bash
npm install
npm run dev
```
